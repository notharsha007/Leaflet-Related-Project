      Project Workflow

      1. Add Markers

      Turn marker mode on with the marker toggle button. Left-click on the map to add markers. Right-click an existing marker to remove it while marker mode is on. Left-click a marker to select it.

      2. Set Grouping Threshold

      Use the Grouping Threshold input to define the maximum distance, in meters, from a seed marker. A marker becomes a group candidate when it is within this threshold from the seed.

      3. Group Markers

      Click Group Markers to run the greedy grouping process. The app checks each remaining marker as a possible seed, finds all remaining markers within the threshold, and selects the seed that creates the largest candidate group.

      4. Understand Greedy Grouping

      The algorithm chooses the best seed for the current step, creates that group, removes those markers from the remaining pool, and repeats. This is greedy behavior, so it is understandable and fast, but not guaranteed to be globally optimal. If two seeds create equal-sized groups, the earlier marker wins the tie.

      5. Inspect Group Results

      Group Results shows the final groups as g1, g2, g3, and so on. Markers and group visuals use matching colors so the map and result list stay connected.

      6. Inspect Seed Candidates

      After grouping, click Show Seed Candidates Details. Each group section lists the markers inside that final group. For each marker, Group Candidates shows all markers within threshold if that marker were used as the seed. Ideal Seed means the marker selected by the greedy algorithm for that group.

      7. Use Radius Rings

      Toggle radius rings to show the threshold radius around the chosen group seeds. These colored rings help explain why nearby markers were included in a group.

      8. Other Controls

      Undo removes the most recently added marker. Clear removes all markers and group visuals. Basemap changes the map style. Show Marker Details displays marker coordinates and group assignment.
